</p>

**See who verified Extreme Demons, right on the level page.**

Uses the [AREDL](https://aredl.net/) database.

Just open any Extreme Demon and the verifier name shows up below the creator. Click the YouTube icon to watch the verification video. On two-player levels, click the label to toggle between Solo and 2P.

- **"Verified by: [Name]"** — level is on the list
- **"[Solo] Verified by: [Name]"** — solo completion for a two-player level
- **"[2P] [Names]"** — two-player completion
- **"Not on AREDL"** — level isn't listed
